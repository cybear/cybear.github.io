If you live in Øresund area (around Copenhagen / Malmö), check out our meetup group - The Øresund Frontend Meetup. Our next meetup will be a presentation by me - Mobile webapps with Phonegap.

Problem: Fragmentation. We need to code Dalvik Java for Android, Objective C on iOS. And then there's Blackberry, Windows Phone 7, Symbian, WebOS by HP/Palm...

Solution: Webapps! :D

HTML5 is not done until 2022 but we can cheat and use it today! Much of the same code can be intensely reused inbetween lots of different devices and platforms. Not to mention possible reuse for a future facebook app, Chrome extension...

The presentation will be about:

  • Advantages and disadvantages of HTML5 webapps vs native smartphone apps.
  • What goodies are available in Mobile Webkit and via the API?
  • How to target many different devices with same code?
  • CSS tricks for different resolutions.
  • How to build a phonegap project for iOS.
  • Which are the mobile JavaScript frameworks to choose from?
  • Phonegap build service in the cloud! Supports Android, WebOS, Symbian, Blackberry so far.

Don't hesitate to join The Øresund Frontend Meetup - it's free and awesome!
